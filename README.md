# 🦾 Zero 机械臂控制系统

> 一个专为 Windows 优化的六轴机械臂控制平台。集成 3D 可视化、AI 视觉识别、大模型对话。

---

## ⚡ 快速启动 (所有用户)

### 选项 1：Docker 部署 (推荐)
1. **获取项目**：`git clone https://github.com/1694037135/zero-robotic-arm.git`
2. **启动**：进入 `robot-control-system` 目录，双击运行 **`deploy.bat`**。
3. **完成**：访问 [http://localhost](http://localhost)。

### 选项 2：本地开发部署
1. **安装依赖**：双击运行 **`1-安装依赖.bat`**。
2. **启动项目**：双击运行 **`2-启动项目.bat`**。
3. **完成**：访问 [http://localhost:3000](http://localhost:3000)。
