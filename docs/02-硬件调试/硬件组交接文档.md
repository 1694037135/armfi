# 🔧 硬件组快速接管手册

本文档专为硬件同学编写，目标是让你以最快速度控制机械臂实体。

---

## ⚡ 核心流程：3分钟连通硬件

### 1. 硬件准备
- 使用 USB 串口线连接机械臂与电脑。
- 确认 COM 号：右键“此电脑”-> 管理 -> 设备管理器 -> 端口 (记下 COM3/COM4 等)。

### 2. 配置修改 (config.json)
打开 `robot-control-system/ai-service/config.json`，修改以下两项：
```json
"SERIAL_ENABLED": true,
"SERIAL_PORT": "COM3"  // 填入你记下的 COM 号
```

### 3. 一键启动
双击运行 **`2-启动项目.bat`**。
- 打开浏览器访问 [http://localhost:3000](http://localhost:3000)。
- 界面右上角点击 **“实物”** 按钮。
- **验证**：按键盘 WASD，机械臂动了，即说明连接成功。

---

## 🛠️ 独立调试工具
如需单独测试硬件而不启动整个系统，请运行 **`debug-tools.bat`**：

1. **[1] 串口监视器**：实时看角度回传，确认硬件心跳。
2. **[3] 标定助手**：用于“点击抓取”功能的视觉校准。
3. **[5] 扫描串口**：自动帮你找机械臂在哪。

---

## 📷 摄像头标定 (视觉组必看)

如果你需要机械臂能“看”到物体并抓取，请按此步骤：

1. **打印标记**：运行 `debug-tools.bat` 选择 [2] 生成 ArUco 标记并打印。
2. **固定位置**：摄像头固定后**千万不能移动**。
3. **运行标定**：选择 [3] 标定助手，按 `a` 检测，按 `空格` 记录坐标。
4. **验证**：在 Web 界面尝试点击画面，机械臂能准确移动到点击位置即成功。

---

## ❓ 硬件常见坑点
- **串口拒绝访问**：关闭其他串口软件 (如 Arduino IDE、串口调试助手)。
- **角度不动**：检查串口线是否松动，或者波特率 (默认 115200) 是否匹配。
- **死机/停止**：直接双击运行 **`3-停止所有服务.bat`** 重启即可。
