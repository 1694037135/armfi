# 🏁 功能测试指南 (一键验证)

> **核心提示**：不需要机械臂也能测试！默认 **模拟模式 (Simulation)** 即可验证所有 AI 和视觉逻辑。

---

## 🚀 1. 快速启动
1. 双击 **`2-启动项目.bat`**。
2. 浏览器访问：[http://localhost:3000](http://localhost:3000)。
3. 确认左上角显示“系统在线”。

---

## ✅ 2. 测试任务清单

### 任务 A：基础指令 (响应 <100ms)
在左侧对话框输入以下关键词，观察 3D 模型：
- [ ] **点头** / **挥手** / **复位**
- [ ] **跳舞** / **转圈**
- [ ] **再来一次** (验证多轮对话记忆)

### 任务 B：YOLO 视觉识别
1. 点击右侧“摄像头”按钮，翻转/选择你的摄像头。
2. 对着摄像头展示杯子、手机或键盘。
3. 输入：**“你看到什么了？”**
- [ ] 预期：AI 准确列出画面中的物品。

### 任务 C：智能语义 (需配置 API Key)
输入非固定关键词的指令：
- [ ] **“帮我活动一下关节”**
- [ ] **“先点头，然后转个圈”**

---

## 🔧 常见故障自查 (Windows)
- **对话没反应**：检查是否运行了 `2-启动项目.bat`，窗口别关。
- **摄像头打不开**：确认没有其他程序 (如微信视频) 占用摄像头。
- **AI 报错**：检查 `config.json` 里的 Key 是否有效，网络是否需要代理。
