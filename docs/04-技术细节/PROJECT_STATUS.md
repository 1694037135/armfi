## 🎯 项目概述

Zero 机械臂控制系统是一个集成 3D 可视化、AI 视觉识别、大模型对话和逆运动学规划的六轴机械臂控制平台。

---

## ✅ 已完成功能

### 核心控制系统
- ✅ **3D 可视化界面**
  - Three.js 渲染机械臂模型
  - 实时关节角度显示
  - 键盘控制（WASD + QE）
  - 模拟/实物模式切换

- ✅ **逆运动学（IK）系统**
  - 目标位置自动规划
  - 碰撞检测
  - 路径平滑优化
  - 工作空间限制

- ✅ **串口通信**
  - 硬件角度回传
  - 实时遥测广播（WebSocket）
  - Mock 模式支持
  - 可配置波特率和协议

### AI 功能
- ✅ **YOLO 视觉识别**
  - YOLOv8 物体检测
  - 实时视频流处理
  - 目标标注和追踪

- ✅ **摄像头标定**
  - 4 点标定算法
  - 像素坐标到机械臂坐标变换
  - 点击抓取功能

- ✅ **大模型对话**
  - Gemini API 集成
  - 自然语言理解
  - 预定义动作库（挥手、舞蹈等）

### 部署方案
- ✅ **Docker 容器化**
  - 三服务架构（Frontend + Backend + AI Service）
  - Docker Compose 一键部署
  - 自动化脚本（deploy.bat/sh）

- ✅ **本地开发支持**
  - Python + Node.js + Java 分离运行
  - 热重载开发模式

## ⚠️ 已知问题与限制

### 当前限制

1. **硬件兼容性**
   - 仅测试了特定舵机型号
   - Windows 串口热插拔需手动重启服务
   - Docker 中串口设备映射仅支持 Linux

2. **摄像头标定**
   - 需要手动固定摄像头位置
   - 透视畸变较大时精度下降
   - 不支持动态相机

3. **性能**
   - YOLO 实时检测需要较高 CPU/GPU
   - WebSocket 消息频率受网络影响
   - 大角度运动时可能出现抖动

### 已知 Bug

| 问题 | 状态 | 影响 |
|------|------|------|
| 快速连续点击可能导致 IK 计算队列堆积 | 🐛 待修复 | 中 |
| Docker 环境下摄像头延迟较高 | 🐛 待修复 | 低 |
| 某些浏览器语音识别不支持中文 | ⚠️ 已知限制 | 低 |
