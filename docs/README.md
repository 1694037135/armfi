## 🛠️ 工具和脚本

项目提供了一系列调试工具，位于 `tools/` 目录：

| 工具 | 用途 | 命令 |
|------|------|------|
| 串口监视器 | 实时查看串口通信 | `python tools/serial_monitor.py --port COM3` |
| ArUco 生成器 | 生成标定标记 | `python tools/generate_aruco_marker.py --sheet` |
| 标定助手 | 摄像头标定 | `python tools/camera_calibration_helper.py` |
| 硬件测试套件 | 自动化测试 | `python tools/hardware_test_suite.py --port COM3` |

**快速启动工具菜单**：
```bash
# Windows
debug-tools.bat

# Linux/Mac
./debug-tools.sh
```

## 🎓 学习路径

### 路径 1：快速体验（1 小时）

1. 阅读 [部署指南](本地部署指南.md) → Docker 部署部分
2. 启动系统：`deploy.bat` 或 `./deploy.sh`
3. 访问 http://localhost
4. 尝试键盘控制（WASD + QE）
5. 测试预定义动作

### 路径 2：硬件集成（3-5 天）

1. 完成路径 1
2. 阅读 [硬件组交接文档](硬件组交接文档.md)
3. 连接串口，配置参数
4. 适配串口协议（如需要）
5. 完成摄像头标定
6. 运行硬件测试套件

### 路径 3：深入理解（1-2 天）

1. 阅读 [技术实现详解](技术实现详解.md)
2. 阅读 [系统架构流程图](系统架构流程图.md)
3. 查看源代码
4. 尝试修改和扩展功能
