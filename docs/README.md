---

## 🚀 快速启动

### 方式 1：Docker 部署 (推荐)
如果你已安装 Docker，这是最快的方式。

1. **获取代码**：`git clone https://github.com/YOUR_USERNAME/zero-robotic-arm.git`
2. **启动**：进入目录，双击运行 **`deploy.bat`**
3. **完成**：访问 [http://localhost](http://localhost)

### 方式 2：本地直接启动
适合需要修改代码或未安装 Docker 的用户。

1. **安装环境**：双击运行 **`1-安装依赖.bat`** (仅首次，自动安装库)
2. **启动项目**：双击运行 **`2-启动项目.bat`**
3. **完成**：访问 [http://localhost:3000](http://localhost:3000)
