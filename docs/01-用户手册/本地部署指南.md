# 🚀 Zero 机械臂一键部署指南

本项目支持 Windows 一键启动。请根据你的习惯选择 Docker 模式或本地模式。

---

## 🐳 模式 A：Docker 部署 (极力推荐)

**适用**：不想配置复杂的运行环境，只想快速运行。

### 1. 准备工作
- 安装并启动 [Docker Desktop](https://www.docker.com/products/docker-desktop/)。

### 2. 启动步骤
1. **进入文件夹**：`robot-control-system/`。
2. **运行脚本**：双击 **`deploy.bat`** (或 `3-启动项目-Docker版.bat`)。
3. **设置 Key**：按照脚本提示或编辑 `config.json` 填入 `GEMINI_API_KEY`。

**访问地址**：[http://localhost](http://localhost)

---

## 💻 模式 B：本地开发模式

**适用**：需要修改代码，或者不想使用 Docker。

### 1. 前置环境
只需安装以下三个软件：
1. **Python 3.10+** (勾选 Add to PATH)
2. **Node.js 18+**
3. **Java 17+** (仅在需要运行后端核心逻辑时使用)

### 2. 启动步骤 (按顺序运行)
1. **安装**：双击 **`1-安装依赖.bat`** (首次运行或代码更新后运行)。
2. **启动**：双击 **`2-启动项目.bat`** (会自动打开前端和 AI 服务窗口)。
3. **后端**：如需使用 Java 后端，在 IDEA 中运行 `RobotControlApplication.java`。

**访问地址**：[http://localhost:3000](http://localhost:3000)

---

## ⚙️ 关键配置 (config.json)

位于 `ai-service/config.json`，核心参数如下：

| 参数 | 说明 | 必填 |
|------|------|------|
| `GEMINI_API_KEY` | 大模型密钥 (Google AI Studio) | 是 |
| `SERIAL_ENABLED` | 是否连接实体机械臂 (true/false) | 否 |
| `SERIAL_PORT` | 串口号 (如 COM3) | 否 |

---

## ❓ 常见问题 (Windows 专用)

- **脚本一闪而过**：请确认已安装 Python/Node 并重启过电脑。
- **端口被占用**：运行 **`3-停止所有服务.bat`** 强制关闭旧进程，然后重试。
- **GEMINI 连接超时**：检查是否设置了 `HTTP_PROXY` (如 Clash 默认 http://127.0.0.1:7890)。
